Infrastructure Packages

Reusable, battle-tested, production-ready solutions for common infrastructure problems.

What’s an infrastructure package?

At Gruntwork, we’ve taken the thousands of hours we spent building infrastructure on Amazon Web Services (AWS) and condensed all that experience and code into pre-assembled Infrastructure Packages. Each Package is a reusable, tested, documented, configurable, best-practices definition of a piece of Infrastructure, written using a combination of Terraform, Go, and Bash, and proven in production at a number of companies.
