Patcher will help you update your codebase all the way from Terrafrom 0.11 to Terraform 1.0, and it supports incremental upgrades (e.g., 0.11 -> 0.12 -> 0.13 -> 0.14 -> 0.15 -> 1.0). Patcher will scan your codebase and open a pull request, such as the one shown in the screenshot below, that transforms your code to use the new version of Terraform. For example, Terraform 0.12 had a particularly painful backwards incompatible upgrade, but Patcher can make such upgrades as painless and smooth as reviewing and merging a PR. At Gruntwork, we have created patches to automatically do each of the incremental upgrades.
